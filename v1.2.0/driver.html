Drivers

This pack provides the following CMSIS Drivers:

  • Ethernet MAC - Ethernet Media Access Control interface driver
  • GPIO - General-Purpose Input/Output Interface driver
  • I2C - Inter-Integrated Circuit driver
  • MCI - Memory Card Interface driver
  • SPI - Serial Peripheral Interface driver
  • USART - Universal Synchronous/Asynchronous Receiver/Transmitter driver
  • USB Device - Universal Serial Bus Device driver
  • USB Host - Universal Serial Bus Host driver
